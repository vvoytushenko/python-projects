Description

The HR boss needs to know something about the employees. Find out the answers to the following questions:

  1. What are the departments of the top ten employees in terms of working hours? Output a Python list of values;
  2. What is the total number of projects on which IT department employees with low salaries have worked? Output a number;
  3. What are the last evaluation scores and the satisfaction levels of the employees A4, B7064, and A3033? Output a Python list where each entry is a list of values of the last evaluation score and the satisfaction level of an employee. The data for each employee should be specified in the same order as the employees' IDs in the question above. Apply the .loc method of pandas to answer the question!
