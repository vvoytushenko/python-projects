Description

A bar chart is helpful if you want to plot the distribution of values across several categories. In this case, let's plot a grouped bar plot for each Nobel Prize category to compare the shares of male and female Nobel laureates.

Objectives

Generate the following grouped bar plot:

  1. Set figure size to (10, 10);
  2. The bar centers are moved by 0.2 from the tick center, the width of bars is 0.4;
  3. The axis font size is 14, the plot font size is 20.
