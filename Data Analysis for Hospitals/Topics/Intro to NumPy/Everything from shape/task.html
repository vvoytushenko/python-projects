Everything from shape?

The only thing you know about a NumPy array a is its shape:

print(a.shape)  # (3, 2, 6, 1)

What else can you infer from this knowledge?

Do you know:

  • its number of dimensions?
  • its size?
  • its length?

In the text field below, write three answers separated by whitespaces: either the value if this info can be inferred from the shape, or the word no otherwise.

For example, your answer may look like 4 no 2, and it will mean that:

  • the array a definitely has 4 dimensions,
  • the size of the array cannot be inferred from the given shape,
  • the length of the array must be 2.

If you are not sure about the answer, revise the information from the "Learning sizes" part in the theory. If it is still hard for you to determine the number of dimensions, the size and the length, you might try to create such an array and write a little code in your coding environment.
